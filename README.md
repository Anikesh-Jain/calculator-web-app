# Calculator Web App

A responsive calculator web application built using HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations
- Interactive calculator interface
- Responsive design
- Clean user interface
- Real-time input handling
- Keyboard/button interaction support

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Purpose

This project was created to practice JavaScript DOM manipulation, event handling, and frontend application logic through a real-world calculator implementation.

## Learning Outcomes

Through this project, I practiced:

- DOM manipulation
- JavaScript event handling
- Input validation
- UI interaction logic
- Responsive frontend design
- Application state handling

## Future Improvements

- Scientific calculator mode
- Keyboard shortcuts
- Theme switcher
- Calculation history
- Advanced mathematical operations
